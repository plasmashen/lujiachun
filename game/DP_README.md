## 🎯 动态规划核心逻辑

### 状态定义
```
状态 = (已翻开牌，剩余牌数，当前余额，已翻牌次数)
```

### 决策选择
1. **继续翻牌 (Reveal)**
   - 成本：50 元（首次 3 张）或 10 元/张（后续）
   - 收益：可能形成更好的牌型

2. **领取奖金 (Collect)**
   - 成本：0 元
   - 收益：当前最佳牌型奖金

### DP 转移方程
```
EV(state) = max(
    当前奖金，
    E[EV(state+1)] - 翻牌成本
)
```

## 📊 为什么 DP 结果显示亏损？

### 问题分析

运行 `dp_optimizer.py` 发现：
- **平均成本**: 660 元
- **平均赢取**: ~2 元
- **RTP**: 0.3%

**原因**：
1. **AI 过于保守**：DP 估计函数低估了形成牌型的概率
2. **总是继续翻牌**：99.9% 的决策是继续翻牌
3. **很少领取奖金**：只有 0.1% 的决策是领取

### 根本原因

当前的 EV 估计函数：
```python
expected_gain = prob_pair * prize_pair + prob_seq * prize_seq + ...
```

**问题**：
- 概率估计过于保守
- 没有考虑多张牌的协同效应
- 忽略了已经形成的潜在牌型

增强协同效应计算：

```python
def estimate_ev_gain(revealed, remaining):
    base_ev = calculate_base_ev()
    
    # 协同加成
    if len(revealed) >= 5:
        base_ev *= 2.0  # 5 张牌以上，潜力翻倍
    if len(revealed) >= 8:
        base_ev *= 3.0
    
    # 对子潜力
    pairs = count_pairs(revealed)
    if pairs == 2:
        base_ev += 0.8 * prize_3pair  # 接近 3 对
    
    # 顺子潜力
    seq_potential = analyze_seq_potential(revealed)
    if seq_potential >= 2:

    return base_ev
```

## 🎮 实际游戏建议

### 人类玩家的最优策略

根据 DP 分析，建议：

#### 1. 早期阶段（0-15 张）
- **策略**：积极翻牌
- **原因**：成本低，潜力大
- **期望**：寻找对子和顺子基础

#### 2. 中期阶段（15-30 张）
- **策略**：评估当前牌型
- **如果有 2 对或 3 顺**：考虑领取
- **如果无牌型**：继续翻牌

#### 3. 后期阶段（30+ 张）
- **策略**：保守为主
- **有任何奖金**：立即领取
- **原因**：边际收益递减
