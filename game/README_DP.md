# 🤖 DP 最优策略版说明

## 📋 核心特性

### 动态规划决策
游戏使用**动态规划 (DP)** 实时计算最优决策：
- **继续翻牌** vs **领取奖金**
- 基于当前牌型、剩余牌数、奖金设置
- 实时计算期望收益 (EV)

---

## 🎮 游戏规则

### 付费规则 ✅
- **首次**: 50 元看 3 张
- **后续**: 10 元/张

### 游戏流程
1. 支付 50 元，翻开 3 张牌
2. AI 显示最优策略建议
3. 选择继续翻牌或领取奖金
4. 重复直到牌用完或余额不足

---

## 🤖 DP 策略逻辑

### 决策公式
```
继续翻牌 EV = 
  对子潜力 × 概率 + 
  刻子潜力 × 概率 + 
  顺子潜力 × 概率 + 
  特殊牌潜力 × 概率 + 
  协同效应系数 × 递减因子
```

### 决策规则
- **继续翻牌**: 当 `继续 EV - 成本 > 当前奖金 × 30%`
- **领取奖金**: 当 `当前奖金 ≥ 继续 EV × 30%`

## 💡 策略建议

### 早期阶段 (0-10 张)
- ✅ **积极翻牌**
- 成本低，潜力大
- 即使有小奖也可继续

### 中期阶段 (10-25 张)
- ⚠️ **根据牌型决策**
- 有 2 对/3 顺：考虑领取
- 无牌型：继续翻牌

### 后期阶段 (25+ 张)
- ✅ **保守为主**
- 有任何奖金立即领取
- 边际收益递减

## ⚙️ 自定义奖金

### 操作步骤
1. 点击"奖金设置"标签
2. 调整各牌型金额
3. 点击"保存设置"
4. AI 策略自动更新

### 推荐配置

#### 公平模式 (RTP≈100%)
```
所有奖金 × 3
```

#### 娱乐模式 (RTP≈150%)
```
所有奖金 × 4.5
```
