# 🀄 49.9 麻将欧欧乐

## 📋 游戏规则

### 付费规则
- **首次看牌**: 50 元看 3 张
- **后续看牌**: 10 元/张
- **初始资金**: 1000 元

### 游戏流程
1. 支付 50 元，翻开 3 张牌
2. 后续每支付 10 元，翻开 1 张牌
3. 从已翻开的牌中选择 2-7 张组成牌型
4. 组合成功获得对应奖金
5. 消除已匹配的牌，继续游戏
6. 目标：最大化收益！
